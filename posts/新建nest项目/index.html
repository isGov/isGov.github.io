  2. 新建entitit映射数据库

看到数据库自动生成了test表就说明成功改了

需求分析

  1. 登录 => 用户注册
  2. 考题选择 => 获取考试题目(查)
  3. 开始 = > 修改试卷（改） ，创建试卷（增）
  4. 考试记录 => 获取试卷列表(查)
  5. 个人中心 => 修改用户 信息(改)
  6. 考题管理 => 新增考试题目
  7. 课程管理 = > curd课程
  8. 学员管理 => 改用户
  9. 阅卷 => 获取试卷内容

数据库表设计

  • exam_exam //考试表
  • exam_subject // 课程表
  • exam_topic // 题目表
  • exam_users //用户表
    • created: Date //创建时间
    • name: string // 名称
    • vChat: string // 微信名字
    • phone: string // 手机
    • avatar: string // 头像
    • graduation time: Date //毕业时间
    • money: number //现在薪资
    • role: string, // 角色
    • id: string,
    • has _person_info: boolean,//是否填写个人信息
    • topic_role:[] //课程权限
    • edu: string //学历
    • techstack: string//技术栈
